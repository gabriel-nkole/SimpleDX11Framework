## SimpleDX11Framework
by Gabriel Nkole

## Description
My test framework for 3D & 2D Rendering using DirectX (Direct3D & DirectInput).

## Features
- 3D Rendering
- Skybox
- Texturing
- 2D Rendering/Animation (Sprites)
- Floor Grid with 1m x 1m squares
- Camera Controls
- Fullscreen
